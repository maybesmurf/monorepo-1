# Doggo Likes Dogs

Shocker, software for doggy sports needs to know what a dog even looks like in the first place.

| Property | Data Type | Notes |
| :--- | :----: | :---- |
| id | uuid | Primary key for the dog table. |
| Call Name | string | This is the typical spoken name of the dog. |
| AKC Breed | enum | AKC official breed of the dog. We will have to get this information from AKC itself. |
| Dog Sex | enum | `Male` or `Female`. Is there a more appropriate hobbyist's word to use? |
| Dog Birthdate | date |  |
| Dog Birthplace | enum | We want to store the dog's birthplace as a country. |
| AKC Registered Name | string | This is not necessarily the spoken name of the dog but rather a formal name. Kind of like how in Mexican culture people will go by two names but their family name includes like 10 names. |
| AKC Title Prefix | enum | This is a list that comes from AKC. |
| AKC Title Suffix | enum | This is a list that comes from AKC. |
| AKC Registration Type | enum | `AKC Reg Number`, `AKC PAL Number`, `Foreign Reg Number` |
| Registration Number | string |  |
| Breeder's Name | string |  |
