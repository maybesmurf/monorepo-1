Ring a Ding Ding

A Ring is an arena where a course is set up and dogs do their runs. It's analagous to a baseball field. Some trials have one ring, with one course going during the day. Others have two rings operating simulataneously.

PropertyData TypeNotes
Ring NamestringDefault to 1 and 2
Classes OfferedenumStandard, Jumpers With Weaves, Time 2 Beat, FAST are the four classes or events that are offered.
Levels available for StandardenumNovice A, Novice B , Open,Excellent, Master ,Novice Preferred,Open Preferred,Excellent Preferred,Master Preferred, Premier , Premier Preferred

Levels available for Time 2 BeatenumTime 2 Beat, Time 2 Beat Preferred
Levels available for FASTenumNovice A, Novice B , Open,Excellent, Master ,Novice Preferred,Open Preferred,Excellent Preferred,Master Preferred
Running orderarray?Each ring needs an order in which these runs will happen, after the entries have been collected and the trial is closed. For example: in this ring, Standard goes first, starting at 8:00AM. The dogs in standard run "Tall to Small", so the Tallest dogs run first, then down the line. After all Standard class runs are complete, Jumpers With Weaves Class goes, Tall to Small, and so on. This running order needs to be adjustable, both by moving around whole Classes, and individual dog runs within a class
Each Height for each Level for each Class needs a Standard Course TimestringWhat is the Standard Course time for this particular course? Example: 69 seconds. This is determined by the judge the day of the trial, determined by using a roller to measure the distance of the course, and this number is used for scoring. Kinda like a PAR score in golf.

Each class is it's own sort of distinct "event" within the day. For each of these classes, we need:

PropertyData TypeNotes
Each Class needs a start timestringExample: Regular starts at 8AM. Jumpers With Weaves starts at 10AM, and so on.
Each class needs Judge assignedJudgeThis is the judge that's umpiring in this ring. Very important - a judge cannot judge more than 350 runs in a day. It's not common that it's close, but that's a rule
Each class needs Heights offeredenumAll, 8,10,12,14,16,18,20,22,24. Vast majority of the time All is offered
Each class needs Levels offeredenum
